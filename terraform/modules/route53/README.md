# CloudSight Route53 Module

## Purpose

Manages Route53 hosted zones and ACM DNS validation records.

## Resources

- Route53 Hosted Zone (optional)
- Route53 DNS Validation Records

## Inputs

- project_name
- environment
- common_tags
- domain_name
- create_hosted_zone
- hosted_zone_id
- validation_records

## Outputs

- hosted_zone_id
- validation_record_fqdns
