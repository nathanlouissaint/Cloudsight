# CloudSight Compute Module

## Purpose

Creates the application compute layer for CloudSight.

## Resources

- EC2 Instance
- Target Group Attachment

## Inputs

- project_name
- environment
- public_subnet_ids
- ec2_security_group_id
- target_group_arn
- instance_profile_name
- instance_type
- common_tags

## Outputs

- EC2 instance information
