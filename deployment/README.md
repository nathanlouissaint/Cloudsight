# CloudSight Deployment Metadata

## Purpose

This directory stores deployment runtime metadata generated by the deployment automation.

These files are intentionally excluded from Git because they represent operational state rather than source code.

---

## Files

### history.log

Append-only deployment journal.

Example:

2026-07-01T18:42:12Z | DEPLOY | sha-2fd9d12

2026-07-01T19:13:47Z | ROLLBACK | sha-a8bc91d

---

### rollback.log

Detailed rollback activity.

Example:

2026-07-01T19:13:47Z

Rolling back

Target Image:

sha-a8bc91d

Verification

Passed

---

### last-deployment.json

Current deployment metadata.

Example

{
  "imageTag": "sha-a8bc91d",
  "operation": "DEPLOY",
  "timestamp": "2026-07-01T18:42:12Z"
}

---

## Operational Notes

These files are generated automatically.

Do not edit them manually.

Deleting them does not affect infrastructure.

They will be recreated automatically during the next deployment.
